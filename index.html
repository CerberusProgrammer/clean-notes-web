Clean Notes - Aplicación de notas con Markdown

Clean Notes es una aplicación para tomar notas que te permite organizar tus ideas de forma eficiente. Características principales: - Editor de Markdown con vista previa en tiempo real - Organización de notas en libros personalizables - Soporte para español e inglés - Modo oscuro y claro - Búsqueda rápida de notas - Diferentes vistas para organizar tus notas - Interfaz minimalista enfocada en la escritura
